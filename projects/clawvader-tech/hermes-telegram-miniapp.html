overview

Hermes Telegram Mini App is a React-based terminal interface that allows users to manage and interact with Hermes agents directly within Telegram. It functions as a mobile-first workspace that connects to a local Hermes installation via a secure gateway, providing a dark-mode dashboard for chat and system monitoring. The application features integrated tools for managing cron jobs, spawning background agent instances, and performing vision-based tasks through local OCR and LLM servers. Security is maintained through dual HMAC-SHA256 and Ed25519 validation to ensure only authorized Telegram users can access the agent.

  • Terminal-style chat with streaming responses and slash command support
  • Real-time system monitoring for CPU, memory, and disk usage
  • Background agent spawning with support for five concurrent instances
