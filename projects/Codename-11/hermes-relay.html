overview

Hermes-Relay is a native Android companion application designed to provide a mobile interface for the Hermes agent platform. It establishes a secure connection between a mobile device and a Hermes server using a Python-based plugin and WebSocket Secure (WSS) communication. Users can interact with their AI agents through a dedicated chat interface, access a remote terminal, and execute device control commands. The system simplifies setup through QR-based pairing and includes a dashboard for monitoring connected devices and bridge activity.

  • Native Android client supporting chat, terminal, and device control
  • Secure WebSocket communication with QR-based server pairing
  • Server-side Python plugin for seamless hermes-agent integration
