breakdown

nesquena / hermes-webui

This zero-build web interface provides a three-panel layout that mirrors the full Hermes Agent CLI experience for both desktop and mobile browsers. It allows users to manage sessions, chat with the agent, and browse workspace files through a lightweight Python and vanilla JavaScript architecture.

outsourc-e / hermes-workspace

A native web workspace that unifies chat, terminal access, and a memory browser into a single interface for managing agent skills. It features real-time SSE streaming to visually render agent tool calls and supports diverse local backends like Ollama and vLLM.

dodo-reach / hermes-desktop

A native macOS workspace that utilizes direct SSH connections to interact with the agent without requiring local file mirroring or daemons. It provides a profile-aware environment featuring a multi-tabbed terminal with customizable themes and live color controls.

xaspx / hermes-control-interface

This self-hosted dashboard acts as a multi-agent gateway for configuring and monitoring multiple agent profiles simultaneously. It stands out for its detailed token analytics, which track costs by model and platform, alongside integrated cron management and system metrics.

fathah / hermes-desktop

Designed as a companion app, this tool simplifies the initial setup of Hermes Agent through guided installation and dependency resolution. It bridges the agent with external messaging platforms like Discord and Telegram while supporting a wide array of LLM providers.

sanchomuzax / hermes-webui

This monitoring-focused dashboard specializes in process oversight, offering real-time tracking of gateway status and session costs. It includes a SQLite-powered full-text search for inspecting conversation histories and a read-only interface for managing scheduled cron jobs.

Euraika-Labs / pan-ui

A comprehensive AI workspace that features an integrated marketplace for discovering and installing new skills and MCP servers. It streamlines deployment with an automated setup wizard that supports daemon and systemd execution for persistent agent profiles.

pyrate-llama / hermes-ui

Featuring a glassmorphic design, this single-file React interface requires no npm dependencies for deployment. It includes a dedicated live artifact panel for rendering HTML and SVG code blocks alongside a real-time system performance monitoring dashboard.
