Project Breakdown

hermes-webui

This lightweight web interface offers full feature parity with the Hermes CLI through a streamlined three-panel layout for chat, sessions, and file browsing. Built with Python and vanilla JavaScript, it provides a responsive, mobile-friendly experience without the complexity of modern build tools.

hermes-workspace

A comprehensive native web workspace that unifies chat, terminal access, and memory browsing into a single interface. It features real-time SSE streaming for visual tool-call rendering and supports diverse local backends including Ollama and vLLM.

hermes-control-interface

This self-hosted dashboard functions as a multi-agent gateway, allowing users to configure profiles and monitor system metrics in real-time. It stands out for its granular security, offering role-based access control with 28 distinct permissions and detailed token cost analytics.

hermes-desktop

A native macOS workspace that provides direct SSH connectivity to Hermes without requiring local file mirroring or background daemons. It includes a multi-tabbed terminal with live theme controls and a profile-aware interface for managing remote sessions and cron jobs.

hermes-desktop

This desktop companion simplifies the initial setup of Hermes Agent through guided installation and dependency resolution. Beyond local chat, it serves as a configuration hub for connecting agents to messaging platforms like Discord and Telegram.

hermes-webui

Focused on observability, this dashboard provides real-time monitoring of session activity, gateway status, and operational costs. It enables deep inspection of agent history through full-text SQLite search and a read-only view of scheduled cron jobs.

pan-ui

A self-hosted workspace that emphasizes agent isolation and extensibility through a built-in marketplace for skills and MCP servers. It features an automated setup wizard that supports daemon and systemd execution for persistent background operation.

hermes-ui

This glassmorphic web interface utilizes a unique single-file React architecture that requires no build steps or external dependencies. It provides real-time log streaming and tool-call visualization alongside an integrated memory inspector for rapid agent configuration.
