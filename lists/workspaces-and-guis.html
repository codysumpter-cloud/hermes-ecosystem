breakdown

nesquena / hermes-webui

This zero-framework interface provides a mobile-responsive three-panel layout for seamless chat, file browsing, and session management. It maintains full parity with the Hermes CLI, offering a lightweight Python and vanilla JavaScript alternative for users who want the complete agent experience in a browser.

outsourc-e / hermes-workspace

A security-hardened workspace that features real-time SSE streaming and integrated rendering for tool calls. It excels at scale, providing specialized interfaces to manage over 2,000 skills alongside a dedicated memory browser and execution approval prompts.

dodo-reach / hermes-desktop

A native macOS application that utilizes direct SSH connections to interact with Hermes without requiring local file mirroring or background daemons. It features a profile-aware workspace and a multi-tabbed terminal with customizable themes for a professional developer experience.

xaspx / hermes-control-interface

This dashboard acts as a multi-agent gateway, allowing users to start, stop, and configure multiple profiles from a single self-hosted hub. It stands out for its detailed token analytics, which track costs across different models and platforms in real-time.

fathah / hermes-desktop

This desktop companion simplifies the onboarding process by providing guided installation and dependency resolution for the Hermes Agent. It serves as a versatile bridge, supporting various LLM providers and integrating messaging gateways for platforms like Discord and Telegram.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React SPA designed for the Telegram Mini App ecosystem, featuring a terminal-style chat with slash command support. It allows users to spawn background agents and monitor system health through live CPU, memory, and disk usage gauges.

sanchomuzax / hermes-webui

Focused on observability, this dashboard provides real-time activity monitoring and cost distribution tracking for Hermes sessions. It includes FTS5 search capabilities for inspecting conversation history and a read-only viewer for configuration and cron job management.

Euraika-Labs / pan-ui

A chat-first workspace that integrates a marketplace for skills, MCP servers, and plugins into the agent workflow. It provides isolated profile management to keep sessions, memory, and API keys separate while visualizing tool execution on a live timeline.

pyrate-llama / hermes-ui

This glassmorphic web interface uses a unique single-file React architecture that requires no build step for self-hosting. It features an artifact panel for live rendering of HTML and SVG outputs alongside real-time SSE log streaming for tool call visualization.
