breakdown

nesquena / hermes-webui

This zero-framework interface provides a mobile-friendly, three-panel layout for managing chats, files, and sessions. It maintains full parity with the Hermes CLI while using only Python and vanilla JavaScript for a lightweight, dependency-free experience.

outsourc-e / hermes-workspace

A security-hardened workspace that features real-time SSE streaming and integrated rendering for tool calls. It specializes in large-scale agent management, offering tools to browse memory and oversee a library of over 2,000 skills.

dodo-reach / hermes-desktop

A native macOS application that establishes direct SSH connections to agents without requiring a middle-layer API or remote daemon. It provides a multi-tabbed terminal environment for managing remote cron jobs, skills, and SQLite session data.

xaspx / hermes-control-interface

This self-hosted dashboard acts as a multi-agent gateway for configuring and monitoring various agent profiles. It stands out for its detailed token analytics, which track costs across different models and platforms in real-time.

fathah / hermes-desktop

This desktop companion simplifies the onboarding process through guided installation and dependency resolution for Hermes Agent. It supports a wide range of LLM providers and includes integrated messaging gateways for platforms like Discord and Telegram.

clawvader-tech / hermes-telegram-miniapp

Built as a React-based Telegram Mini App, this mobile-first dashboard enables background agent spawning and live output monitoring. It features a terminal-style chat interface alongside system health gauges for tracking CPU and memory usage on the go.

sanchomuzax / hermes-webui

A monitoring-focused dashboard that provides real-time activity tracking and cost distribution analysis. It features deep conversation inspection powered by FTS5 search and a read-only view for managing scheduled cron jobs.

Euraika-Labs / pan-ui

A chat-first workspace that integrates an execution timeline for tools and a marketplace for skills and MCP servers. It allows for complete isolation between sessions by managing separate profiles, memory stores, and API keys.

pyrate-llama / hermes-ui

This glassmorphic web interface uses a single-file React architecture that requires no build step for deployment. It includes a dedicated artifact panel for live rendering of HTML, SVG, and code generated during agent sessions.
