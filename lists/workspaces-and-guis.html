breakdown

nesquena / hermes-webui

This zero-framework interface provides a mobile-responsive, three-panel layout for managing chats, files, and sessions. It maintains full parity with the Hermes CLI, offering a lightweight Python and vanilla JavaScript alternative for users who want the complete agent experience in a browser.

outsourc-e / hermes-workspace

A security-hardened workspace that features real-time SSE streaming and integrated tool call rendering. It excels at scale, providing dedicated interfaces for managing over 2,000 skills alongside a memory browser and execution approval prompts.

dodo-reach / hermes-desktop

A native macOS workspace that establishes direct SSH connections to agents without requiring a middle-layer API or remote daemon. It provides a multi-tabbed terminal environment and specialized tools for managing remote SQLite session stores and cron jobs.

xaspx / hermes-control-interface

This self-hosted dashboard acts as a multi-agent gateway for starting, stopping, and configuring various agent profiles. It stands out for its detailed token analytics, allowing users to track costs by model and platform while monitoring system metrics and cron jobs.

fathah / hermes-desktop

This desktop companion simplifies the onboarding process with guided installation and dependency resolution for the Hermes Agent. It serves as a configuration hub for various LLM providers and includes integrated messaging gateways for platforms like Discord.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React dashboard designed to run within the Telegram Mini App ecosystem. It features a terminal-style chat interface with slash command support and live system health gauges for monitoring CPU and memory usage on the fly.

sanchomuzax / hermes-webui

Focused on observability, this dashboard provides real-time activity monitoring and cost distribution tracking. It features deep conversation inspection powered by FTS5 search, allowing users to audit history and view scheduled cron tasks in a read-only format.

Euraika-Labs / pan-ui

A chat-centric workspace that integrates a marketplace for skills, MCP servers, and plugins. It utilizes isolated profile management to keep sessions, memory, and API keys separate while providing a visual timeline of tool executions.

pyrate-llama / hermes-ui

This glassmorphic web interface uses a unique single-file HTML architecture powered by React and a Python proxy. It includes a specialized artifact panel for live previews of HTML, SVG, and PDF files generated by the agent during a session.
