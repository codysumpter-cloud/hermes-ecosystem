breakdown

vectorize-io / hindsight

Hindsight implements biomimetic data structures to facilitate long-term learning through dedicated retain, recall, and reflect workflows. It is designed to help agents build evolving mental models and integrates seamlessly with major providers like OpenAI, Anthropic, and Ollama.

garrytan / gbrain

This project functions as a self-wiring knowledge graph that uses automated entity extraction and typed linking to organize agent intelligence. It distinguishes itself through a hybrid search system that combines vector retrieval with backlink-boosted ranking and twenty-six specialized skills for data enrichment.

greyhaven-ai / autocontext

This recursive context harness uses a multi-agent loop to propose, analyze, and curate knowledge for complex tasks. It specializes in self-improvement, distilling validated execution data into optimized runtimes to enhance agent behavior over repeated scenario runs.

stephenschoettler / hermes-lcm

A Lossless Context Management plugin that utilizes a hierarchical DAG-based engine to ensure no message is ever lost during compaction. It features three-level escalation logic and immutable message persistence within profile-scoped SQLite databases for structured history retrieval.

elkimek / honcho-self-hosted

This backend provides a private, self-hosted solution for cross-session memory persistence using Docker, PostgreSQL, and Redis. It is ideal for users prioritizing data sovereignty, supporting any OpenAI-compatible provider or local inference server.

yoloshii / ClawMem

ClawMem is a local-first context engine that uses a unified SQLite vault to manage memory across different agents without cloud dependencies. It employs a sophisticated hybrid retrieval strategy combining vector search, BM25, and graph traversal to automate decision capture and handoffs.

plur-ai / plur

This shared memory layer utilizes an open YAML 'engram' format to store persistent data locally with zero API costs. It features a unique biological approach to memory management, where information automatically decays or strengthens based on its usage frequency.

amanning3390 / flowstate-qmd

An anticipatory memory system that prefetches relevant context by monitoring agent session logs in real-time. Built as an MCP-native server, it leverages SQLite-vec and GGUF models to provide low-latency RAG capabilities for coding agent clients.
