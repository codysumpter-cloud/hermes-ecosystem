Project Breakdown

hindsight

Hindsight implements biomimetic data structures to facilitate long-term learning through dedicated retain, recall, and reflect workflows. It offers a simple two-line LLM wrapper that automates memory management, allowing agents to evolve their mental models over time.

gbrain

This opinionated brain provides 25 specialized skills for autonomous data ingestion and enrichment alongside over 30 MCP tools. It utilizes PGLite for zero-config local storage, combining vector and relational data to serve as a robust foundation for Hermes Agent integrations.

autocontext

This recursive context harness uses a multi-agent loop to extract and persist validated knowledge across eleven distinct scenario families. It is particularly effective for distilling stable agent behaviors into local models via MLX integration, ensuring success in complex or fragile tool-based tasks.

hermes-lcm

This plugin introduces a lossless context engine that uses immutable SQLite storage and a hierarchical DAG structure to ensure no conversation data is ever deleted. It provides built-in tools for searching and expanding compacted context, enabling multi-level summary escalation for long-running sessions.

honcho-self-hosted

Honcho provides a private, self-hosted backend for cross-session memory that supports any OpenAI-compatible provider or local inference server. The included automated setup script simplifies the deployment of Docker, PostgreSQL, and Redis for a secure memory infrastructure.

ClawMem

ClawMem is a local-first context engine that eliminates cloud dependencies by using a hybrid retrieval system combining vector search, BM25, and multi-graph traversal. It enables seamless memory sharing between Hermes, OpenClaw, and Claude Code without requiring external API keys.

plur

This shared memory layer stores persistent data as open-format YAML engrams, utilizing a hybrid retrieval system with zero API costs. It features a unique biological approach where memories automatically decay or strengthen based on their usage frequency within multi-agent systems.

flowstate-qmd

Developed for the Hermes 2026 Hackathon, this project features an anticipatory cache that prefetches context into a local intuition file. It leverages SQLite-vec and node-llama-cpp to provide a privacy-focused, local-first memory solution via an MCP server.
