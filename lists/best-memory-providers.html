breakdown

garrytan / gbrain

An opinionated brain for Hermes Agent featuring a self-wiring knowledge graph with automated entity extraction. It combines vector retrieval with backlink-boosted ranking and includes 26 specialized skills for media enrichment.

vectorize-io / hindsight

A biomimetic memory system that implements retain, recall, and reflect workflows for long-term learning. It simplifies integration via a two-line LLM wrapper for automatic memory management.

greyhaven-ai / autocontext

A recursive context harness that uses multi-agent loops to extract and persist validated knowledge across complex tasks. It can distill stable behaviors into local models through MLX integration.

stephenschoettler / hermes-lcm

A lossless context management plugin utilizing a hierarchical DAG-based engine for multi-level compaction. It ensures data integrity via immutable-first SQLite storage so no message is ever deleted.

elkimek / honcho-self-hosted

A private, self-hosted backend that enables cross-session persistence for Hermes Agent. It supports local inference servers or OpenAI-compatible providers and deploys via Docker, PostgreSQL, and Redis.

yoloshii / ClawMem

A local-first context engine that uses a unified SQLite vault for on-device memory persistence. It employs a hybrid retrieval strategy combining vector search, BM25, and graph traversal.

plur-ai / plur

A shared memory layer for multi-agent systems that stores persistent data as open-format YAML engrams. It features zero-cost hybrid retrieval and a system for memory decay and strengthening based on usage.

amanning3390 / flowstate-qmd

An anticipatory memory system that prefetches context into a local intuition file for faster retrieval. Built for privacy with SQLite-vec and node-llama-cpp, it integrates as an MCP server for coding workflows.
