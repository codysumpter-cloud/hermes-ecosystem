breakdown

builderz-labs / mission-control

This self-hosted orchestration platform manages multi-agent workflows using a lightweight SQLite backend with zero external dependencies. It is ideal for Hermes users who require real-time task monitoring and strict quality control through the built-in Aegis review system.

swarmclawai / swarmclaw

Swarmclaw provides a self-hosted runtime for building autonomous agent swarms that support over 20 providers, including local Ollama and Hermes. It distinguishes itself by enabling agents to learn new skills from conversations and maintain persistent memory across delegated tasks.

howdymary / hermes-agent-metaharness

This framework focuses on meta-optimization by using an automated outer-loop search to refine the harness code used for Hermes Agent evaluations. It orchestrates complex benchmarks like TBLite and TB2, tracking performance frontiers through structured candidate mutation.

1ilkhamov / opencode-hermes-multiagent

This project coordinates a massive pipeline of 17 specialized agents dedicated to tasks ranging from infrastructure to security. It enforces rigorous quality gates by routing all outputs through mandatory reviewer and tester agent loops to ensure production-ready code.

runtimenoteslabs / gladiator

Gladiator creates a competitive arena where two autonomous AI companies, powered by Hermes Agent, compete for GitHub stars. The system orchestrates nine specialized agents and visualizes their real-time execution, code generation, and scoreboard metrics on a live dashboard.

supermodeltools / bigiron

Bigiron implements an AI-native software development lifecycle by providing Hermes with structural code graphs via the Supermodel MCP. It automates an eight-phase development process, using graph-diff gates to ensure architectural integrity at every step from planning to health checks.

Rainhoole / hermes-agent-acp-skill

This framework utilizes ACP-style delegation to route tasks between Hermes subagents and external models like Codex or Claude Code. It prioritizes operational safety by isolating delegation workflows to prevent context pollution and enforcing strict timeouts and output limits.
