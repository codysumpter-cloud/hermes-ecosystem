breakdown

builderz-labs / mission-control

This self-hosted orchestration platform manages multi-agent workflows across frameworks like CrewAI and AutoGen with zero external dependencies. It is essential for Hermes users who require a private environment to monitor task spend and implement human-in-the-loop sign-offs via the Aegis review system.

swarmclawai / swarmclaw

Swarmclaw provides a self-hosted runtime for building autonomous agent swarms that support 23 different model providers, including Hermes and local LLMs. It distinguishes itself through persistent agent memory and a unique 'conversation-to-skill' learning mechanism that expands agent capabilities over time.

howdymary / hermes-agent-metaharness

This specialized framework focuses on meta-optimization by using an automated outer-loop search to improve the performance of Hermes Agent on benchmarks. It orchestrates structured candidate mutations to find the most efficient harness code for TBLite and TB2 evaluation tasks.

runtimenoteslabs / gladiator

Gladiator creates a competitive arena where two autonomous AI companies, powered by Hermes Agent, compete for GitHub stars in real-time. The system orchestrates nine specialized agents and visualizes their tasks, code generation, and scoreboard metrics through a live dashboard.

1ilkhamov / opencode-hermes-multiagent

This framework coordinates 17 specialized agents across distinct development phases ranging from research and planning to infrastructure. It ensures high-quality outputs by implementing mandatory quality gates and code reviews at every stage of the multi-agent lifecycle.

supermodeltools / bigiron

Bigiron implements an AI-native software development lifecycle by providing Hermes with structural code graphs via the Supermodel MCP. It enforces architectural integrity through an eight-phase process where each transition is protected by automated graph-diff phase gates.

Rainhoole / hermes-agent-acp-skill

This project enables ACP-style delegation by routing tasks between Hermes subagents, Codex, and Claude Code while preventing parent context pollution. It prioritizes safety and efficiency through isolated workflows, configurable timeouts, and strict output character limits.
