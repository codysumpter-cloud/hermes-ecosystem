breakdown

numtide / llm-agents.nix

This project provides automated daily updates for AI coding agent packages, including Hermes, through the Nix ecosystem. It enables instant execution via Nix flake commands, allowing developers to run the latest agent versions without manual installation or configuration.

rookiemann / portable-hermes-agent

Designed for Windows users, this portable desktop agent requires no installation, admin rights, or Docker to provide a full GUI and over 100 integrated tools. It features a built-in workflow engine for scheduled automation and local model management with GPU monitoring.

TheAiSingularity / hermesclaw

Hermesclaw focuses on high-security deployments by providing hardware-level enforcement of network, filesystem, and process boundaries. It features three security policy presets and secure gateways for integrating Hermes with communication platforms like Telegram and Discord.

0xrsydn / nix-hermes-agent

This NixOS module enables declarative management of Hermes configurations, documents, and systemd services for fully reproducible deployments. It prioritizes security through sops-nix integration for secrets handling and supports a hybrid skill model for both managed and interactive installs.

xmbshwll / hermes-agent-docker

This repository provides a streamlined Docker sandbox that bundles the official Hermes installation with mini-swe-agent. It supports persistent state through mounted directories and allows users to target specific Hermes releases using build arguments.

JackTheGit / hermes-autonomous-server

Targeting production environments, this project configures the Hermes Gateway as a reboot-safe systemd service for headless Ubuntu and Debian servers. It is optimized for autonomous execution, enabling the agent to perform recurring AI cron jobs without manual intervention.

ellickjohnson / portainer-stack-hermes

This deployment option uses a Docker Compose stack to provide a web-based terminal interface for Hermes via ttyd. It is designed for easy management through Portainer and utilizes GitHub Actions for automated image builds.
