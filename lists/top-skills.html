Project Breakdown

Anthropic-Cybersecurity-Skills

This library provides 754 structured cybersecurity skills mapped to major industry frameworks like MITRE ATT&CK and NIST. It equips Hermes with specialized expertise across 26 security domains, making it a powerful asset for defensive and offensive security workflows.

avoid-ai-writing

This skill audits and rewrites content to eliminate 36 distinct AI writing patterns and structural tells. By utilizing a three-tier word replacement system and a second-pass audit, it ensures Hermes produces more natural, human-like prose.

wondelai/skills

A modular, cross-platform library that brings frameworks for architecture, UX design, and product strategy to agentskills.io-compatible agents. It allows Hermes to quickly scale its capabilities through a CLI-driven marketplace of professional-grade skillsets.

pydantic-ai-skills

This project implements the Agent Skills open specification for Pydantic AI with a focus on efficiency. It uses progressive disclosure of metadata to reduce token usage while supporting both filesystem-based and programmatic Python skill definitions.

drawio-skill

This skill enables Hermes to generate editable .drawio diagrams from natural language, complete with automated self-correction for layout issues. It includes specialized presets for machine learning models, including tensor shape annotations and multiple export formats.

litprog-skill

This literate programming tool allows Hermes to weave runnable code and documentation together into synchronized .lit.md files. It features a 'tangler' to extract source files and automatically syncs edits back to the primary prose document.

chainlink-agent-skills

Designed for Web3 workflows, this skill provides standardized interactions for the Chainlink Runtime Environment and CCIP. It allows Hermes to discover contracts, monitor cross-chain sends, and manage decentralized oracle network tasks using the agentskills.io spec.

icarus-plugin

Icarus provides Hermes with shared memory across instances via markdown-based directories to track long-term work. It includes an end-to-end pipeline for extracting decision-quality data to train and evaluate replacement models.

skilldock.io

This registry offers an OpenAPI-driven SDK for discovering and managing reusable AI skills with recursive dependency resolution. It also features integrated commerce tools, allowing for the monetization of skills via TON and Stripe.

black-forest-labs/skills

The official skill for FLUX image generation, providing Hermes with standardized prompting guidelines and API integration patterns. It supports precise hex color specifications and JSON structured prompting to ensure high-fidelity visual outputs.

hermes-skill-factory

This meta-skill monitors active workflows to automatically detect and propose new, reusable skills. It generates both Python scaffolds and documentation, allowing Hermes to self-evolve by codifying its own successful patterns.

maestro

Maestro is a harness for long-running tasks that facilitates structured context transfers between agents using UKI v5.4 handoffs. It maintains shared state on disk and provides a TUI for real-time mission tracking and plan approval.

execplan-skill

This skill enables autonomous execution of complex, multi-step tasks with built-in checkpoints for progress tracking. It ensures reliability by providing rollback procedures for safe recovery if a task encounters an error.

hermeshub

A community-driven hub for browsing and searching Hermes skills with one-click installation via the CLI. It prioritizes safety with automated security scanning for prompt injections and supports a creator marketplace via the Micropayment Protocol.

bmad-module-skill-forge

This forge converts existing repositories and developer discourse into verified agentskills.io-compliant skills using AST analysis. It ensures high accuracy by linking every AI instruction directly to specific source code line numbers.

Agentic-MCP-Skill

A progressive Model Context Protocol (MCP) client that uses three-layer lazy loading to minimize token consumption. It maintains persistent connections to multiple servers via a socket-based daemon for efficient tool and schema inspection.

skillsdotnet

This project brings the agentskills.io specification to the C# and .NET ecosystem with integrated MCP support. It provides resource templates for on-demand skill loading and vendor-specific shortcuts for environments like Cursor and Copilot.
