breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This project provides 754 structured cybersecurity skills mapped to industry standards like MITRE ATT&CK and NIST CSF 2.0. It enables Hermes to perform specialized tasks across 26 domains, including malware analysis and cloud security, using the agentskills.io standard.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate detectable AI patterns through a tiered word replacement system and a two-pass verification process. It is specifically designed to help Hermes Agent produce more human-like, undetectable prose.

wondelai / skills

A modular cross-platform library that equips agents with frameworks for architecture, UX design, and product strategy. It features a dedicated CLI for easy installation of specialized professional skillsets into any agentskills.io-compatible environment.

DougTrajano / pydantic-ai-skills

This library implements the Agent Skills open specification for Pydantic AI with a focus on type safety and efficiency. It uses progressive disclosure of metadata to significantly reduce token consumption during skill discovery.

Agents365-ai / drawio-skill

This skill allows Hermes to generate complex draw.io diagrams, including UML and architecture maps, directly from natural language. It features grid-aligned layouts and specialized tensor shape annotations for machine learning visualizations.

smartcontractkit / chainlink-agent-skills

Focused on blockchain interoperability, this project provides specialized skills for interacting with Chainlink's Oracle networks and CCIP. It allows Hermes to perform on-chain operations and smart contract interactions using the official Agent Skills specification.

tlehman / litprog-skill

This skill introduces literate programming capabilities, allowing Hermes to weave runnable code and documentation into unified markdown files. It includes a 'tangler' to extract source files and automatically syncs edits between code and prose.

esaradev / icarus-plugin

Icarus provides Hermes with a shared memory system and a pipeline for training its own replacement models. It extracts decision-quality data from active workflows to facilitate fine-tuning and cross-instance knowledge persistence.

chigwell / skilldock.io

A comprehensive registry and SDK for managing the lifecycle of reusable AI skills with built-in monetization via Stripe and TON. It automates dependency resolution and provides a Python-based CLI for local skill installation.

black-forest-labs / skills

This is the official integration for FLUX image generation, providing Hermes with standardized prompting guidelines and API patterns. It supports precise visual control through JSON structured prompting and hex color specifications.

Romanescu11 / hermes-skill-factory

This meta-skill acts as a workflow observer that automatically detects and proposes new, reusable skills based on the agent's activity. It generates both the necessary Python scaffolds and documentation to turn repetitive tasks into permanent capabilities.

ReinaMacCredy / maestro

Maestro is a local-first harness for long-running missions that utilizes structured memory and validation assertions. It enables Hermes to manage complex plans with milestones and perform clean handoffs between agent sessions.

tiann / execplan-skill

This skill enables autonomous execution of multi-step tasks with built-in checkpoints and real-time progress tracking. It ensures operational safety by providing rollback procedures to recover from errors during complex workflows.

cablate / Agentic-MCP-Skill

A progressive Model Context Protocol client that uses three-layer lazy loading to minimize token usage. It maintains persistent server connections via a socket-based daemon and supports hot-reloading of configurations.

armelhbobdad / bmad-module-skill-forge

This tool converts source code and developer documentation into verified, version-pinned skills for AI agents. It links every instruction to specific file paths and line numbers to ensure high-fidelity function calling.

amanning3390 / hermeshub

A community-driven hub for browsing and installing Hermes skills with integrated security scanning for prompt injections. It supports micropayments for premium skills and features an agent-to-agent feedback system for trust scores.

PederHP / skillsdotnet

This project brings the agentskills.io specification to the .NET ecosystem, allowing C# developers to integrate Hermes with MCP. It supports automatic skill discovery and uses progressive disclosure to optimize context window efficiency.
