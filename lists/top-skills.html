breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to industry standards like MITRE ATT&CK and NIST CSF 2.0. It enables Hermes to operate across 26 security domains, including malware analysis and cloud security, using the agentskills.io integration standard.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate detectable AI patterns using a comprehensive 109-entry word replacement table. It features a two-pass audit system compatible with Hermes Agent to ensure text remains natural and bypasses automated detection.

wondelai / skills

A cross-platform library that provides modular skills for architecture, UX design, and product strategy via a dedicated CLI. It allows Hermes to quickly adopt professional frameworks for marketing and business logic through an npx-based installation process.

Agents365-ai / drawio-skill

This skill enables Hermes to generate complex draw.io diagrams, including UML and architecture flows, from natural language descriptions. It supports grid-aligned layouts and specialized tensor shape annotations for machine learning visualizations.

DougTrajano / pydantic-ai-skills

This project implements type-safe support for the Agent Skills open specification specifically for Pydantic AI. It optimizes Hermes' performance by using progressive disclosure of metadata to significantly reduce token consumption during skill discovery.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose new, reusable skills for Hermes. It generates both Python scaffolds and documentation, allowing the agent to programmatically expand its own toolkit through observation.

tlehman / litprog-skill

This literate programming skill allows Hermes to weave runnable source code and prose documentation into unified .lit.md files. It features an automated tangler that extracts code for execution and syncs edits back to the primary documentation.

esaradev / icarus-plugin

This plugin introduces cross-instance shared memory, allowing Hermes to remember past work and extract decision-quality training data. It creates an end-to-end pipeline for fine-tuning replacement models based on the agent's own historical performance.

smartcontractkit / chainlink-agent-skills

Built on the agentskills.io spec, this library allows Hermes to interact with the Chainlink network for cross-chain transfers and data feed monitoring. It provides the necessary tools for smart contract testing and secure oracle operations within an agentic workflow.

ReinaMacCredy / maestro

A harness for long-running missions that provides Hermes with structured memory and milestone-based planning. It uses a local-first state management system to handle handoffs between agent runs via detailed markdown briefs.

chigwell / skilldock.io

A registry and SDK for managing the lifecycle of reusable AI skills with built-in support for Stripe and blockchain-based monetization. It automates dependency resolution and local installation, providing a structured marketplace for Hermes capabilities.

black-forest-labs / skills

The official skill set for FLUX image generation, offering standardized prompting guidelines and API integration patterns for Hermes. It supports precise hex color specifications and structured JSON prompting to ensure high-fidelity visual outputs.

tiann / execplan-skill

This skill enables Hermes to execute complex, multi-step tasks with autonomous tracking and real-time decision logging. It includes built-in rollback procedures and checkpoints to ensure safe recovery during long-running operations.

amanning3390 / hermeshub

A community hub for browsing and installing Hermes skills with automated security scanning for prompt injection and data exfiltration. It features a proof-of-use review system and supports micropayments for premium skill developers.

armelhbobdad / bmad-module-skill-forge

This utility converts existing repositories and developer documentation into immutable, version-pinned skills for Hermes. It maps AI instructions directly to source code line numbers, preventing documentation rot in agentic workflows.

cablate / Agentic-MCP-Skill

A progressive MCP client that utilizes three-layer lazy loading to minimize context window usage for Hermes. It maintains persistent server connections via a socket-based daemon and supports hot-reloading of configurations without downtime.

PederHP / skillsdotnet

This project brings the agentskills.io specification to the .NET ecosystem, allowing Hermes to interface with C# based MCP tools. It implements progressive disclosure to save tokens while enabling automatic skill discovery from standard agent directories.
