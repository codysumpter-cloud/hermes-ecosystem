breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to major industry frameworks like MITRE ATT&CK and NIST. It equips Hermes with specialized domain knowledge across 26 security sectors, ensuring high-fidelity performance in defensive and offensive security tasks.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate 36 distinct AI writing patterns and structural tells. By using a three-tier word replacement system and a second-pass audit, it helps Hermes produce more natural, human-like prose that bypasses linguistic detection.

wondelai / skills

A modular, cross-platform library that brings standardized skills for architecture, UX design, and product strategy to Hermes. It utilizes the agentskills.io specification, allowing for easy installation of professional-grade capabilities via a dedicated CLI.

DougTrajano / pydantic-ai-skills

This project implements the Agent Skills open specification for Pydantic AI with a focus on efficiency. It uses progressive disclosure of metadata to reduce token usage while supporting both programmatic and filesystem-based skill definitions.

Agents365-ai / drawio-skill

This skill enables Hermes to generate complex draw.io diagrams, including UML and system architectures, from natural language descriptions. It features six specialized presets and supports grid-aligned layouts with animated connectors for professional visual documentation.

smartcontractkit / chainlink-agent-skills

Designed for blockchain interaction, this skill set allows Hermes to manage Chainlink Runtime Environment and CCIP workflows. It follows the agentskills.io spec to facilitate cross-chain sends, contract discovery, and real-time monitoring.

tlehman / litprog-skill

This literate programming skill allows Hermes to weave runnable code and prose together into synchronized .lit.md files. It includes a tangler to extract source files and automatically syncs edits between documentation and codebases.

esaradev / icarus-plugin

Icarus provides Hermes with a self-memory system that uses markdown-based directories to share context across different instances. It features an end-to-end pipeline for extracting decision-quality data to train and evaluate replacement models.

chigwell / skilldock.io

This registry acts as a discovery and management hub for reusable AI skills using an OpenAPI-driven Python SDK. It supports recursive dependency resolution and includes integrated commerce features for monetizing skills via TON and Stripe.

black-forest-labs / skills

The official skill set for FLUX image generation, providing Hermes with precise prompting guidelines and API integration patterns. It supports structured JSON prompting and hex color specifications to ensure high-quality, predictable visual outputs.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose new, reusable skills. It generates both the necessary SKILL.md documentation and Python scaffolds, allowing Hermes to programmatically expand its own capabilities.

ReinaMacCredy / maestro

Maestro is a harness for long-running agents that maintains shared state on disk and facilitates context transfers via structured handoffs. It provides a read-only TUI and JSON snapshots to track complex missions and cross-feature learning.

tiann / execplan-skill

This skill enables Hermes to execute complex, multi-step tasks with built-in checkpoints and recovery procedures. It tracks decisions in real-time and provides autonomous rollback capabilities to ensure safe execution of long-running processes.

cablate / Agentic-MCP-Skill

A progressive Model Context Protocol client that uses three-layer lazy loading to minimize token consumption for tool definitions. It maintains persistent connections to multiple MCP servers via a socket-based daemon for efficient metadata retrieval.

armelhbobdad / bmad-module-skill-forge

This forge converts existing repositories and developer documentation into agentskills.io-compliant instruction sets. It maps AI instructions directly to source code line numbers, allowing Hermes to ingest and utilize specific library versions accurately.

amanning3390 / hermeshub

HermesHub is a community marketplace for browsing and installing skills with one-click CLI integration. It prioritizes safety by performing automated security scans for prompt injections and malicious payloads on all hosted skills.

PederHP / skillsdotnet

This project brings the agentskills.io specification to the C# and .NET ecosystem, providing MCP resource templates for on-demand loading. It includes vendor-specific shortcuts to ensure Hermes remains compatible with .NET-based development environments.
