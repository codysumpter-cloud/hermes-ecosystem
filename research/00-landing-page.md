# Hermes Agent — Landing Page

**URL:** https://hermes-agent.nousresearch.com/  
**Tagline:** "An Agent That Grows With You"

## Overview

Hermes Agent is an autonomous, open-source agent framework from Nous Research. Unlike traditional coding copilots or chatbot wrappers, it's "an autonomous agent that lives on your server, remembers what it learns, and gets more capable the longer it runs."

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
hermes setup
```

## Core Features

- **Multi-Platform Integration** — Telegram, Discord, Slack, WhatsApp, Signal, Email, CLI, and more
- **Adaptive Learning** — Persistent memory and auto-generated skills
- **Automated Scheduling** — Natural language cron scheduling
- **Parallel Processing** — Isolated subagents with separate conversations/terminals
- **Robust Sandboxing** — 5 execution backends (local, Docker, SSH, Singularity, Modal)
- **Extended Capabilities** — Web search, browser automation, vision, image gen, TTS, multi-model reasoning
