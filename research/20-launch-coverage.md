# Hermes Agent — Launch Coverage & Press

## Launch Timeline

- **February 2026:** Nous Research releases Hermes Agent. MIT licensed. Collected 22,000 GitHub stars and 242 contributors within weeks.
- **March 2026:** v0.4.0 "Platform Expansion Release" — OpenAI-compatible API server, 6 new messaging adapters, 4 new providers, MCP server management with OAuth 2.1, 200+ bug fixes.
- **April 2026:** v0.7.0 — Pluggable memory providers, credential pool rotation, anti-detection browser backend, inline diff previews.
- **Current (April 8, 2026):** 36,400+ stars.
